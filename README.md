# Darth Maul Portfolio Website

## Overview

This project is a responsive portfolio website themed around **Darth Maul** from *Star Wars*.  
The purpose of the site is to demonstrate core front-end development skills, with emphasis on **semantic HTML**, **responsive design**, **CSS/Bootstrap usage**, and **version control with GitHub**.

The content theme is secondary; the primary focus is on **technical implementation and code quality**.

## Page Descriptions

### Home
- Introduces the site and theme
- Uses a Bootstrap navbar and hero section
- Fully responsive with collapsible navigation

### About
- Provides background information
- Uses semantic sections and Bootstrap list groups
- Includes Font Awesome icons for visual clarity

### Portfolio
- Displays portfolio items using a repeatable card structure
- Built with Bootstrap grid and cards
- Easily extensible by duplicating `<article>` elements
